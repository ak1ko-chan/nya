Бинарный поиск

Он позволяет на возрастающей или убывающей функции находить значение за О(log)

Идея:

Делим область функции на две половины и смотрим где может лежать наше значение и сужаем границы

while (r - l > 1) {
int m = (r + l) / 2;
if (check(m))
r = m;
else
l = m;
}
